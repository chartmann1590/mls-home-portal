Resilient Search Pipeline

Direct MLS (optional) plus scrape/API fallback chain designed for non-ideal real-world sources.

AI Affordability + Fit

Uses local LLM support to evaluate affordability and generate listing-level fit summaries.

Docker-First Deployment

Single compose stack with health checks, restart policy, and persisted saved data volume.

Quick Start

  1. Clone the repository.
  2. Copy .env.example to .env.
  3. Run docker compose up -d --build.
git clone <your-repo-url>
cd <repo-folder>
cp .env.example .env
docker compose up -d --build

Repository Highlights

  • FastAPI backend with structured search/detail/saved endpoints
  • Frontend search UI with listing detail modal and save actions
  • SQLite persistence for saved search criteria and saved homes
  • GitHub Actions CI, Pages deploy, and secret scanning workflows
